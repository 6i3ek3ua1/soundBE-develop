# 🚀 БЫСТРЫЙ СТАРТ - SoundBE

## За 5 минут до работающей системы

### ⚙️ Требования
- Python 3.9+
- 8GB RAM минимум (16GB рекомендуется)
- 6GB VRAM если есть GPU (рекомендуется)
- ~8GB свободного места на диске

### 1️⃣ Установка (3 минуты)

```bash
# Клонируем проект
git clone <repo-url>
cd soundBE-develop

# Создаём виртуальное окружение
python -m venv venv

# Активируем (выберите свою ОС):
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# Устанавливаем зависимости
pip install -r requirements.txt
```

**Первый запуск обычно долгий (~5-10 мин) - загружаются большие модели (~4GB)**

### 2️⃣ Запуск

```bash
# Самый простой способ - с UI
python main.py

# ИЛИ без интерфейса (консоль)
python main.py --no-ui

# ИЛИ с конкретным микрофоном
python main.py --device 1

# ИЛИ на другом порту
python main.py --ui-port 8080
```

### 3️⃣ Использование

1. **Откроется браузер** на `http://localhost:8000`
2. **Выберите микрофон** в разделе "⚙ Настройки"
3. **Нажмите "▶ Начать"** для запуска захвата
4. **Говорите в микрофон** - система будет:
   - ✅ Распознавать вашу речь
   - ✅ Переводить её
   - ✅ Озвучивать результат
5. **Результаты** отображаются в реальном времени
6. **История** сохраняется автоматически

## 🔧 Конфигурация (опционально)

Создайте файл `.init` в корне проекта:

```ini
# Выберите размер Whisper модели
# tiny (быстрая) → base (баланс) ← рекомендуется
# small, medium, large (точнее, медленнее)
WHISPER_MODEL=base
WHISPER_LANGUAGE=en

# Перевод
TRANSLATION_ENABLED=1
TARGET_LANGUAGE=English

# Озвучка
TTS_ENABLED=1
TTS_LANGUAGE=en

# Логирование
LOG_LEVEL=INFO
```

Или скопируйте готовый пример:
```bash
cp .init.example .init
```

## 🐛 Если что-то не работает

### "ModuleNotFoundError: No module named..."
```bash
# Переустановить зависимости
pip install -r requirements.txt --upgrade
```

### "CUDA not found" или "Out of memory"
```bash
# Использовать CPU (медленнее, но работает)
set CUDA_VISIBLE_DEVICES=""  # Windows
python main.py

# Или на Linux/Mac:
export CUDA_VISIBLE_DEVICES=""
python main.py
```

### Микрофон не найден
```bash
# Просмотреть доступные устройства
python main.py --no-ui
# (система спросит номер)

# Или найти номер устройства
import sounddevice as sd
print(sd.query_devices())

# Запустить с нужным устройством
python main.py --device 3
```

### Медленно работает
- Попробуйте `WHISPER_MODEL=tiny` вместо `base`
- Отключите перевод: `TRANSLATION_ENABLED=0` в `.init`
- Закройте другие приложения

### Озвучка не работает
```bash
# Переустановить pyttsx3
pip install --upgrade pyttsx3

# Проверить голоса
python -c "import pyttsx3; e=pyttsx3.init(); print([v.name for v in e.getProperty('voices')])"
```

```
   ↓
Озвучка результата (pyttsx3)

+ Статус системы в реальном времени
```

| Компонент | Размер |

## 🔄 Типичный рабочий процесс

```
1. python main.py

5. Система активирована (индикатор зелёный)
6. Говорите по русски
7. Система:
   - Распознаёт текст (0.5 сек)

   - Озвучивает (0.1 сек)

9. Результат сохраняется в истории
10. Нажимаете "Остановить" когда готово
```

## 📊 Первый запуск: что происходит

```
1. Python стартует
2. Загружаются конфиги (мгновенно)
3. Инициализируется логирование (мгновенно)
4. Создаётся Orchestrator (мгновенно)
5. Запускается Eel с браузером (~1-2 сек)
6. Открывается localhost:8000 (~1 сек)
   ↓
   При первом захвате:
7. Загружается Whisper модель (~2-3 сек)
   ↓
   При первом переводе:
8. Загружается Qwen2 модель (~3-5 сек)
   ↓
   При первой озвучке:
9. Инициализируется pyttsx3 (~0.5 сек)

Total первого раза: ~10 сек
Следующие разы: ~1-2 сек полный цикл
```
